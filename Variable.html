    루비 변수

    Variable¶ ↑

    지역 변수

    me = '나형준'

    지역 변수는 소문자로 시작

    상수

    VERSION = '0.0.1'

    상수는 대문자로 시작

    인스턴스 변수

    @name = '유이냥'
     

    attr_accessor :name

    이런 방식으로 클래스 내부의 인스턴스 변수 선언 가능

    전역 변수

    $here = '조선'

    $를 변수 이름 앞에 써주면 전역 변수 선언 가능
